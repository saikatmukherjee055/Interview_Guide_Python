# Unix Practice Lab

A ready-to-run dataset and workbook for practicing `grep`, `awk`, `find`, and common Unix utilities.

## Files

- `QUESTIONS.md` — 120 exercises without answers
- `SOLUTIONS.md` — complete commands, explanations, and expected outputs
- `UNIX_PRACTICE_WORKBOOK.md` — questions with collapsible solutions
- `FILE_INDEX.txt` — complete file/directory inventory
- `reset_metadata.sh` — restores timestamps and permissions used by `find` problems

## Start

```bash
cd unix_practice_lab
./reset_metadata.sh
less QUESTIONS.md
```

Use a second terminal or open `SOLUTIONS.md` only after attempting a problem.
